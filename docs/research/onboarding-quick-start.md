# Onboarding Quick‑Start (Draft)

## Flow

1. **Pick a folder** (scope is explicit).
2. **Choose a quick‑start task**.
3. Review the plan → approve writes.

## Quick‑Start Cards (v1)

### Default cards (4)

#### 1) Organize Downloads

**Prompt**
“Organize this folder by file type and month. Move old installers into an Archive folder.”

**Best for**: messy Downloads or Desktop folders

---

#### 2) Receipts → Spreadsheet

**Prompt**
“Make a spreadsheet from these receipt photos with date, vendor, and amount.”

**Best for**: expense reports

---

#### 3) Notes → Summary Report

**Prompt**
“Summarize the documents in this folder into a one‑page report.”

**Best for**: research and meeting notes

---

#### 4) Images → PDF

**Prompt**
“Convert these images into a single PDF under 10MB.”

**Best for**: scan batches, receipts, handouts

### Additional cards (optional)

#### Find Duplicates

**Prompt**
“Find duplicate files and move extras to a Duplicates folder.”

**Best for**: photo folders, shared drives

---

#### Clean CSVs

**Prompt**
“Clean all CSVs in this folder: normalize dates and remove duplicate rows.”

**Best for**: weekly exports

---

#### Contracts → Summary Table

**Prompt**
“Summarize each contract with key terms, renewal dates, and obligations.”

**Best for**: legal/admin folders

---

#### CSV → HTML Dashboard

**Prompt**
“Create an HTML dashboard from this CSV with 3 key charts.”

**Best for**: basic analytics

## Notes

- These are **safe, file‑scoped** tasks that don’t require connectors.
- Keep prompts short and outcome‑oriented.
- Hide advanced options by default; expose them via “More”.
